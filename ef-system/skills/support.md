# Skill: @support

## Purpose
Activate the user's personalized psychological and cognitive scaffolding.

## Execution
1. **Context Check:** Read `areas/health/SUPPORT-KB.md` and `ef-system/USER-PROFILE.md`.
2. **Identification:** Match the current situation to a known "Breaking Point" (e.g., Spiral, Overwhelm, Freeze).
3. **Activation:** Propose the intervention defined in the Support KB.
   - *Example:* "I notice you've been searching for 45 minutes. Your KB says this is a 'Research Abyss'. Should we set a 5-minute timer to decide?"
4. **Escalation:** If the user is in a "Shutdown," offer to draft a brief status update for their Social Anchor.
