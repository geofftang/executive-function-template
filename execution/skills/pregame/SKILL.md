# Skill: @pregame — Social Event Prep

## Purpose
Quick pre-event social prep. Surfaces the right principles, techniques, and conversation topics for whatever you're heading into.

## Prerequisites
This skill references a `socializing-principles.md` file in your social area folder (e.g., `areas/[social-area]/socializing-principles.md`). If you don't have one yet, create it with your personal social principles and techniques — or skip Step 1 and work from the event type defaults below.

## Execution

1. **Read** `areas/[social-area]/socializing-principles.md` (if it exists)
2. **Ask event type:**
   - Casual group (party, pre, dinner with friends)
   - Group with new people (event, conference afterparty, friend-of-friend)
   - 1-on-1 (catching up, deepening a friendship)
   - Hosting (you're organizing)
3. **Surface the pre-game card** (your top principles, always)
4. **Surface 3-5 techniques** most relevant to the event type:
   - New people → heavy on initiating + sharing your inner world
   - Existing friends → heavy on deepening/escalating + presence
   - Hosting → also load hosting resources (icebreakers, games, flow management)
   - 1-on-1 → heavy on listening + making them feel valued
5. **Pull conversation topics** from `state/STATE.md` — what's been happening recently (projects, events, life context). No stale list; always fresh.
6. **Optional:** If user mentions specific people attending, suggest relevant topics or things to ask about based on known context.

## Output

```
## Pre-Game

[Top principles card]

### For Tonight ([event type])
- [3-5 relevant techniques]

### Conversation Fuel (Recent Life)
- [3-5 things from STATE.md / recent context]

### One Thing to Practice
- [Pick ONE technique to consciously try tonight]
```

## Compliance
- [ ] Pre-game card surfaced
- [ ] Techniques matched to event type
- [ ] Conversation topics pulled from current state, not a stale list
- [ ] One practice focus identified
