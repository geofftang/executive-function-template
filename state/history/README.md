# History

Weekly review archives. Created by `@weekly-review` (Part F: Archive & Reset).

**File format:** `YYYY-MM.json` — monthly files, weeks appended.

**Structure:**
```json
{
  "month": "YYYY-MM",
  "weeks": [
    {
      "week_range": "YYYY-MM-DD to YYYY-MM-DD",
      "weekly_focus": { ... },
      "sessions": [ ... ]
    }
  ]
}
```

**Usage:** Strategic reviews scan these for quarterly pattern analysis.
